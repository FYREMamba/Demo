Technical Deep Dive

Explore the underlying technology, security protocols, and performance specifications.

Encryption & Privacy

  • End-to-End Encryption: AES-256 for all message content.
  • Key Exchange: ECDH (Elliptic Curve Diffie-Hellman) for secure mesh discovery.
  • Zero-Knowledge Routing: Intermediate nodes cannot read message content.
  • Decentralized Identity: No central authority required for device authentication.

System Requirements & Metrics

  • Message Latency: <500ms direct, <2s mesh routing.
  • Discovery Time: New devices detected within 10 seconds.
  • Battery Impact: <5% additional drain in RECEIVE mode.
  • Scalability: Up to 50 mesh nodes per local network, with core anchors extending range up to 10km.
